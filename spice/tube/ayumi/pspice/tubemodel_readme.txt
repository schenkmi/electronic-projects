変更点(3.10)
・6J7, 6C-A10, 50H-B26, 1619, 8045G の電極間容量が間違っていたので修正しました。

変更点(3.00)
・パラメータの最適化の関数を変更し、
  カットオフの特性がより忠実に再現されるようになりました。
・五極管の内部抵抗値を見直しました。
・国産の真空管(6G-A4, 6R-P15, 6C-A10, 50H-B26, 8045Gなど)、
  ヨーロッパの真空管(PX4, PX25, DA30など)、
  水平偏向管を追加しました。

変更点(2.35)
・6CG7, 6EJ7, 6SJ7, 6SN7, 6V6のデータに誤りがあったため、修正しました。

変更点(2.34)
・プレート-カソード間に10GΩ相当の抵抗成分を入れました。
  これにより、差動、SRPP、カスコードなどで直流的に浮いたノードができなく
  なりました。
・6SN7, 6SJ7を追加しました。

変更点(2.33)
・電極間容量を変更しました。
  特に、多極管のコントロールグリッド-スクリーングリッド間の容量を付加したので、
  三極管接続やウルトラリニア接続において
  ミラー効果による容量が以前より多くなります。
  この変更を多極管の三極管接続モデルにも反映してあります。
・飽和電流を変更しました。
  多極管の肩特性がより正確に表されるようになりました。
・6C33CB を追加しました。

変更点(2.32)
・一部のモデルで、グリッド電圧を非常に高くしたときに
  プレート電流やスクリーングリッド電流が負になるバグを修正しました。

変更点(2.31)
・ノードの数を減らしました。
・モデルを追加しました(125種)。
・ファイル名を *.lib から *.inc に変更しました。
  (SIMetrix用は *.mod です。)
・グリッド電流のモデルを見直しました。
  特に、送信管(211, 801など)・B級用出力管(46, 53など)、
  ポジティブグリッド管(6AC5)で、
  グリッド電流の特性が明示されているものについては、
  Ep=Egの場合のプレート電流とグリッド電流がほぼ再現されるようにしてあります。
・真空管名とモデルの対応表を modelindex.csv として収録しました。
・SIMetrix用のノード数の非常に少ないモデルを別途作成してあります。
  20球程度のシミュレーションができるようです。
  SIMetrix用のモデルに関しては、黒田徹氏のご協力を頂きました。

SIMetrixバージョン使い方のヒント
・モデルを適当なフォルダに解凍します。
・個々のモデルをドラッグ&ドロップしても登録できますが、
  File -> Model Library -> Add/Remove Libraries でフォルダを指定すると、
  そのフォルダの *.mod がすべて検索対象になります。
・File -> Model Library -> Associate Models and Symbols で
  モデルとシンボルを結びつけますが、
  同じ端子配列のモデルは一括して処理できます。
  三極管なら三極管だけを Ctrl+クリック ですべて選択し、
  OK を押してシンボルと結びつけます。
  間違えて五極管を選択するとメッセージが出ますが、作業を続行できます。
・SIMetrixの詳しい使い方は、
  「電子回路シミュレータ　SIMetrix/SIMPLISスペシャルパック」
  http://www.cqpub.co.jp/hanbai/books/38/38311.htm
  等をご覧ください。
